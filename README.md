# 明白纸 mingbai

给中国用户的「身后数字事务」自助整理工具。单文件网页应用，无后端、无依赖、不联网、不收集任何数据。

一句话原理：生前把散落在微信、支付宝、游戏、云相册里的数字家当整理成一份家人看得懂、找得到的清单，配一套「每个平台该怎么办」的操作指南。

## 三不承诺

- **不存密码**——工具里没有任何密码/凭证输入框，也不会问你要
- **不碰资金**——提现、转移只指路，不代办任何资金操作
- **不是遗嘱**——清单不具有法律效力，产权分配请走公证处或遗嘱库

## 产品架构

1. **资产盘点向导**：按 8 类（社交/支付/理财/游戏/云存储/订阅/域名及网站/自媒体账号）逐项登记，每类内置常见平台快捷选项＋自定义添加。每项资产记录：平台名、账号备注（怎么认出这是谁的账号，不写密码）、预估价值档位、处理意愿（留念/注销/转移给家人/提现/由家人决定）、凭证存放线索（只写在哪能找到，不写密码本身）、想对家人说的话。清单视图按类别分组展示，可编辑/删除，顶部显示完成度概览。
2. **家属操作指南**：按平台渲染「家属操作卡」——材料清单、办理步骤、入口、电话、法律依据、信息来源，并如实标注可信度（`verified` 官方渠道确认 / `reported` 据公开报道，办理前请与平台确认 / `unverified` 未经官方核实，仅供参考——视觉权重最重，避免用户误当作官方口径）。已接入两路调研交付的 16 个平台真实数据：银行存款、微信、支付宝、手机号、QQ、抖音、微博、B站、淘宝/闲鱼、京东、网易、腾讯游戏、米哈游、Steam、百度网盘、Apple/iCloud，按家属实际办事优先级排序（资金与验证锚点类账户在前）。列表页顶部有「先看这里」通用提示：先别急着注销手机号、善用《个人信息保护法》第49条近亲属信息权利。
3. **平安确认**（本版不做）：v0.2 之后可选，定期「我还在」确认，连续未确认则提示预先指定的家人「该去找那份清单了」——仅做纯本地提醒，不做任何服务器托管通知。

设计规范见 `DESIGN.md`（继承 anhao 家族 DNA：暖米白底＋茶青主色＋胶囊按钮；红色降级为仅用于删除类不可逆操作；打印样式独立设计，黑白优先）。

## 使用方法

- **双击 `index.html`** 用浏览器本地打开即可使用，无需安装、无需联网。
- **登记资产**：首页「开始盘点」→ 选一个类别 → 点快捷平台或自己填名称 → 填写备注/价值/处理意愿/线索 → 保存。数据自动存入本机浏览器的 localStorage，刷新页面不丢失。
- **导出/导入**：清单视图里「导出 JSON」下载一份带日期文件名的备份；「导入 JSON」可从备份文件恢复（会覆盖当前设备上的数据，导入前有二次确认）。
- **打印**：清单视图「打印这份清单」、指南详情页「打印这张操作卡」，调用浏览器打印，`@media print` 专门排版为黑白的正式文书样式（封面标题、生成日期、逐类列表、末尾免责声明与签名留白区）。

## 隐私说明

纯本地应用：不联网、无统计、无任何网络请求。所有数据（资产清单）只存在当前设备浏览器的 localStorage 里，离开这台设备/清除浏览器数据即消失——想长期保存，请手动导出 JSON 文件或打印纸质件，自行存放（如家里保险柜、加密文件）。

## 与 anhao 的家族关系

明白纸是「家庭暗号」（anhao）的兄弟项目，同属家庭信任工具族：anhao 应对「AI 换脸换声诈骗」这类紧急场景，明白纸应对「生前从容整理身后事」这类从容场景。两者共享暖米白＋茶青的视觉语言与单文件、无后端、本地优先的产品形态，但明白纸刻意收起了 anhao 的 SOS 式紧迫感（详见 `DESIGN.md` 的 Taste 章节）。
